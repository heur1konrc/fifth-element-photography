Clean Image Descriptions

Remove excessive line breaks and formatting issues from image descriptions.

What This Tool Does
  • Removes empty paragraphs with whitespace or &nbsp;
  • Reduces multiple consecutive <br> tags to single breaks
  • Removes excessive whitespace between HTML tags
  • Trims whitespace inside paragraphs
  • Creates automatic backup before cleaning
Note: This operation will create a backup file before making any changes. The backup will be saved with a timestamp in the same directory as the original file.

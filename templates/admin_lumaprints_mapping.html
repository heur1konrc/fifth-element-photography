How Lumaprints Library Mapping Works

  1. Upload to Lumaprints: Manually upload your high-res images to your Lumaprints library via their dashboard
  2. Get Library ID: Copy the library ID from Lumaprints for each uploaded image
  3. Map in Admin: Use this interface to link your gallery images to their Lumaprints library IDs
  4. Enable Print Orders: Once mapped, customers can order prints of those images
